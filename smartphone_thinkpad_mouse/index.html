シミュレーターの操作方法

上のiPhone画面の「赤ポチ」をドラッグして動かしてください。

  • 3Dチルト: 動かす方向に赤ポチが立体的に傾きます。
  • ステップハプティクス: 指を動かす距離に応じて、一定間隔（12px）で心地よいクリック音（ハプティクス代替）が鳴ります。指を止めると音も止まります。
  • クラシック3ボタン: 左右のクリックに加え、中央の青ボタンを押しながらドラッグすると、右側のダミー画面が縦横にスクロール（ナチュラルスクロール補正）します。

スクロールデモエリア

中央の青いスクロールボタンを押しながら赤ポチをドラッグすると、このエリアがスクロールします。ナチュラルスクロールが再現されていることを実感してください。

スクロールの滑らかさは、Bonjour通信を介して送信される高解像度な相対ベクトルによって実現されています。

ThinkPadの赤いドットは、単なるシンボルではなく、指先の力をダイレクトにカーソル速度に変換する極上のセンサーです。

このiOS版では、物理的な変位量（タッチ移動距離）に応じたハプティクスフィードバックを組み合わせることで、今までにない操作フィールを実現しました。

さらにテストテキスト: スクロール可能な領域を十分に確保しています。スクロールボタンを押しながら操作してみてください。

ドラッグ＆ドロップ動作時には、マルチタッチ競合を排除するための特別なジェスチャー処理が行われ、ホールド状態が100%維持されます。

プロダクトの特徴

ただのマウスアプリではない。ThinkPad愛好家がディテールにこだわり抜いたプレミアムな設計。

🔴

極上のTrackPoint UI & アニメーション

指の力と方向に合わせて赤ポチが3Dで傾き、影がリアルに変化。画面のどこをタッチしても自動でそこが支点になり、吸い付くような操作を開始できます。

⚡

物理移動同期ステップハプティクス

時間ベースの振動を廃止し、指の実際の移動距離（12px）に完全にシンクロしたハプティクス（コココという微細な抵抗感）をトリガー。指を止めるとハプティクスも瞬時に停止します。

🤝

完璧なドラッグ＆ドロップ

SwiftUIのジェスチャー競合をバイパスする「UIKitブリッジ型タッチ検出」を新設計。左クリックを押しながら別指で自在にスライドさせてもクリックが絶対に途切れず、快適なドラッグが可能です。

🔄

直感的ナチュラルスクロール

中央の青いスクロールボタンを押しながらのスクロールは、Macのスクロール方向に完全に最適化。短押し（0.35秒未満）時は自動で「ミドルクリック」として認識します。

📶

スマートサイレント自動接続

Bonjourサービスを採用し、同一ネットワーク上のMacを自動探索。前回の接続先を記憶し、起動するだけで自動接続。Macのスリープ復帰時もバックグラウンドで即座に再接続します。

セットアップ方法

Mac側で受信用のヘルパーを実行し、iOSアプリをインストールするだけで即座に使えます。

1

Mac側: ヘルパープログラムの起動

ターミナルを起動し、Mac側ヘルパーのディレクトリへ移動して実行します。

cd macOS_thinkpad_helper
swift run
💡 初回起動時

Macの「システム設定」>「プライバシーとセキュリティ」>「アクセシビリティ」で、ターミナルに対するコンピューターの制御許可が必要です。

2

iPhone側: iOSアプリのビルドと起動

Xcodeを使用してiPhone実機にアプリをインストールします。

  1. iPhoneをMacにUSBケーブルで接続します。
  2. smartphone_thinkpad_mouse.xcodeproj を Xcode で開きます。
  3. 開発ターゲットとして実機（iPhone）を選択し、Run (⌘R) を実行します。
  4. アプリが起動すると、自動的にBonjourで同一ネットワーク上のMacを検出し、自動接続されます。

通信プロトコル仕様

Bonjourによって動的に決定されたポートで、極めて軽量なTCPパケット（JSON形式）を遅延ゼロでやり取りします。

🖱️ マウス相対移動 (Move)

{
  "type": "move",
  "dx": 12.34,
  "dy": -5.67
}

🔄 スクロールホイール (Scroll)

{
  "type": "scroll",
  "dx": 1.5,
  "dy": -3.2
}

🔴 マウスボタン操作 (Button)

{
  "type": "button",
  "buttonType": 1, 
  "isDown": 1
}

buttonType: 1 = 左, 2 = 右, 3 = ミドル
isDown: 1 = 押し下げ, 0 = 離す, 2 = 瞬時クリック
